Project Summary

Goal

The Programming Language Mentions Analysis Project aims to provide a comprehensive analysis of job postings in the data sector, focusing on the programming languages required and job classifications. By leveraging data from Glassdoor, the project seeks to identify the most commonly used programming languages in data roles, classifing job titles into relevant categories. The graphs will illustrate the prevalence of specific programming languages in job postings, enabling users to understand the skills in demand.

Process

First I identified a Kaggle source that displayed 742 data job descriptions from Glassdoor job openings. Then, I downloaded the CSV file and inputted it into a SQL database using Microsoft SQL Service to run the database on a server and Azure Studio to make edits to the data. In Azure, I edited the data, eliminated null values, and separated the job descriptions into different job title columns. Once that was done, I connected my updated SQL database into Tableau and created graphs and storyboards to display and analyze the data.

Highlights

The project was mainly ran on SQL and used Azure Studio to create a server that will run the database. The CSV file was edited and cleaned in SQL in order to obtain the most accurate information for this project.
